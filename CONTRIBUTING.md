# Contributing to ProofStampBot-OSS

Thanks for your interest in improving ProofStampBot-OSS.

## How to contribute

1. Fork the repository
2. Create a feature branch
3. Make focused changes with clear commit messages
4. Open a pull request with a short technical summary

## Development notes

- Use Node.js 20 LTS
- Install dependencies with `npm ci`
- Build with `npm run build`
- Run locally with `npm run start`

## Pull request checklist

- No secrets or private keys committed
- `npm run build` passes locally
- Behavior and config changes are documented in `README.md`
- Changes are minimal and related to one goal
